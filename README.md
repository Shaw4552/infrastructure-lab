# infrastructure-lab

Public, sanitized infrastructure lab documentation demonstrating enterprise-style:
- VLAN segmentation
- Firewall policy design (least privilege)
- Dockerized service architecture
- Secure remote access (VPN)
- Backup strategy & recovery thinking

> Note: This repository uses fictional IP ranges and redacted examples. No real secrets, keys, or identifying infrastructure details are included.

## Goals
- Demonstrate network/security architecture fundamentals
- Show documentation discipline and operational thinking
- Provide reusable templates (compose, scripts, policies)

## Contents
- `docs/` architecture writeups
- `docker/` sanitized compose examples + `.env.example`
- `scripts/` automation samples
- `diagrams/` topology images (sanitized)
